Is programming easy

Is programming easy? Yup. If you use frameworks, I think it is easy to come up with an API and then build the frontend using your favorite frontend javascript framework. For me, the most easy-to-understand framework for frontend is Angular. You need a framework Yes. My explanation is simple: Frameworks make projects more easier to build with in the immediate time-frame. As the project size goes larger, like you become a unicorn startup like what silicon valley raves about, my thinking goes you should have enough profits coming in that it’s worth it to invest in performance improvements.
